# Senior Software Engineer — Enterprise IDE / JVM Tooling

## Seniority
Mid-Senior level. Full-time individual contributor.

## Industry
Engineering and information technology — building a flagship IDE for enterprise Java and Kotlin development.

## Position overview
Develop end-to-end IDE features and subsystems for an enterprise IDE product, focused on enterprise frameworks, build tools, and application servers.

## Key responsibilities
- Take end-to-end ownership of IDE features and subsystems related to enterprise frameworks, build tools, and application servers.
- Design and implement advanced tooling for Java, Kotlin, Spring, and related ecosystems.
- Collaborate with senior engineers and tech leads to maintain architectural consistency.
- Drive innovation and usability improvements in large-scale developer tools.
- Contribute to maintaining quality, reliability, performance, and developer-experience standards.

## Required qualifications
- Strong expertise in Java and Kotlin with deep JVM understanding.
- Solid experience with the Spring ecosystem and other enterprise frameworks.
- Proven track record working on complex systems and large-scale codebases.
- Ability to design, maintain, and evolve clean, extensible, and well-architected software.
- Genuine interest in IDE internals and language tooling.
- Excellent communication and collaboration skills.

## Technical focus areas
- Enterprise frameworks and libraries (Spring, Quarkus, Hibernate).
- Build tools and runtime configurations.
- Application server integration.
- Language tooling and IDE internals.
- Developer-experience optimization.
