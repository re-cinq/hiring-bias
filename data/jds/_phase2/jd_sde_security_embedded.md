# Software Development Engineer — Security Features (Embedded Networking)

## Seniority
Mid-Senior level. Full-time individual contributor.

## Industry
Computer and network security — embedded firmware for LAN-switch products.

## Key responsibilities
- Serve as a member of a Software Development Department responsible for developing LAN-switch products.
- Investigate current and future architectures, requirements, and technology needs in collaboration with internal teams.
- Own one or more security and related software functions or modules within embedded products through the complete development lifecycle: design, coding, testing, and bug fixes.
- Collaborate across software development, hardware development, and quality-assurance teams to deliver high-quality products.
- Actively participate in resolving technical and complex issues.
- Research and implement new technologies and tools relevant to product development.

## Required qualifications
- Minimum 3+ years of software-development experience in IT or software industry.
- Strong proficiency in C programming language for UNIX software development.
- Native-level Japanese language proficiency.
- Ability to read English technical documentation; email communication in English required (spoken English is a plus).
- Experience investigating and analyzing the Linux kernel with source-code modification capability.

## Technical skills and knowledge

### Security technologies
- User-communication encryption and key exchange (e.g., MACsec).
- User-authentication systems (Captive Portal, IEEE 802.1X, MAC authentication).
- Device attack detection and tamper detection.
- Encryption technologies.

### Networking
- Layer-2 and Layer-3 networking technologies.
- Network switch operation and configuration.
- REST API, NETCONF, SNMP, Telemetry, Ansible.

### Additional preferred experience
- Python or other programming languages.
- Development-environment building.
- Linux kernel vulnerability countermeasures.
- Chip-vendor SDKs.
- Hardware control for embedded devices.
- System management for embedded devices.
- UI and business logic for embedded applications.
- Switch chip knowledge.
- BIOS and bootloader technologies.
