# Senior Manager — Digital Solutions (C++ Developer)

## Seniority
Mid-Senior level.

## Industry
Automation machinery manufacturing — on-premises and cloud platform solutions for industrial automation.

## Experience profile
- 8–12 years of experience.
- B.E. / B.Tech / M.E. / M.Tech in a relevant field.

## Role summary
Experienced C++ developer designing, developing, and delivering secure, scalable, high-performance on-premises and cloud platform solutions. Combines hands-on development, architectural design, secure engineering practices, and agile collaboration; cloud-technology exposure is advantageous.

## Key responsibilities
- Architect and build high-performance applications using C++ and STL with strong object-oriented principles.
- Develop modular, loosely-coupled solutions and message-broker-based large-scale applications.
- Create networked, multi-threaded systems ensuring performance, reliability, and scalability.
- Contribute to IoT application development using REST and MQTT interfaces.
- Collaborate on architecture designs, UML diagrams, and technical documentation.
- Follow Secure Development Lifecycle (SDL) practices per IEC 62443-4-1 standards.
- Develop test plans and support validation/integration testing.
- Mentor team members on coding and security standards.

## Core technical competencies
- C++, STL, and object-oriented programming.
- Networking, multi-threading, and concurrent programming.
- Message brokers and event-driven architectures.
- REST APIs and MQTT protocol.
- Secure coding practices and industrial cybersecurity concepts.
- UML, design patterns, and software architecture documentation.
- Agile development methodologies.

## Security expertise

- IEC 62443 industrial-cybersecurity standards.
- Application security risk identification and mitigation.
