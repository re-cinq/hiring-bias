# Junior / Mid-Level Fullstack Developer

## Seniority
Entry level. Full-time individual contributor.

## Industry
Renewable energy / construction technology — a digital platform supporting energy consulting and building-renovation workflows.

## Overview
A fullstack development role focused on building pragmatic, scalable solutions across the entire technology stack. The position emphasizes understanding how products function from database to user interface and taking progressively more responsibility.

## Key responsibilities
- Develop and enhance a platform supporting energy consulting and renovation measures.
- Advance customer and contractor portals for matching energy consultations with construction projects.
- Integrate multiple data sources including CRM systems, document management, and automation tools.

## Required skills
- TypeScript proficiency (basic to advanced level).
- REST API understanding and web architecture knowledge.
- Relational database familiarity, preferably PostgreSQL.
- React and/or Nest.js experience.
- Interest in clean software architecture.
- Strong learning ability and team collaboration skills.

## Experience profile
- First to several years of practical software development through employment, internships, or relevant projects.

## Nice-to-have qualifications
- LLM usage and API integration experience.
- Machine learning exposure (image recognition, OCR, document verification).
- AWS services knowledge (ECS, EKS, RDS, S3, SQS, Terraform).
- Kubernetes experience.

## Work environment
High on-site presence required.
