# Senior Fullstack Developer

## Seniority
Mid-Senior level. Individual contributor.

## Industry
Software development / engineering and information technology — internal automation tooling for an enterprise engineering organization.

## Experience profile
- 6+ years as a Full Stack Engineer or similar role.

## Primary responsibilities
- Design, develop, and maintain end-to-end features across frontend and backend.
- Translate user needs into technical solutions through collaboration with product and design teams.
- Write clean, efficient, and well-documented code.
- Integrate products with internal and external APIs and services.
- Contribute to architecture decisions, scalability, and performance improvements.
- Participate in code reviews to ensure quality and best practices.
- Debug and resolve issues rapidly.
- Optimize applications for speed, usability, and scalability.
- Stay current with emerging technologies and industry trends.
- Improve development processes and practices.

## Primary stack
- TypeScript
- Node.js
- Nest.js
- React
- Vitest

## Secondary technologies
- Kotlin / Java (for select backend services)

## Required technical skills
- Solid engineering and architecture skills; ability to build reliable solutions.
- Proficiency in multiple programming languages (TypeScript, Kotlin, Java, Python).
- Deep understanding of backend and frontend development.
- Experience with relational and/or NoSQL databases (PostgreSQL, MongoDB, MySQL).
- Understanding of cloud platforms (e.g. AWS, GCP, or Azure) and containerization (Docker or Kubernetes).
- Proficiency with Git version control.
- Testing framework expertise (Jest, Vitest, Playwright).
- Authentication, authorization, and security best-practices knowledge.
- Experience with AI agent development.

## Core qualifications
- Bachelor's degree in computer science or engineering, or equivalent hands-on experience.
- Customer-oriented delivery focus; understanding of user needs and requirements.
- Strong problem-solving abilities and attention to detail.
- Excellent communication and collaboration abilities.
