# Senior Software Development Engineer — Routing Tiling

## Seniority
Mid-Senior level. Full-time individual contributor.

## Industry
Software development / engineering and information technology — geospatial / navigation data pipelines.

## Key responsibilities
- Design, build, and operate critical components of a routing data pipeline.
- Transform raw map data into optimized routing tiles suitable for production environments.
- Enhance data freshness and reduce pipeline processing latency significantly.
- Integrate real-time traffic signals and map updates to improve routing accuracy.
- Maintain high reliability and availability of routing data for global users.
- Build automated systems that preserve data integrity and consistency at scale.

## Required skills and experience
- Modern C++ development in commercial or open-source environments, or equivalent Rust experience.
- Knowledge of operating-system and distributed-system concepts (fault tolerance, consistency, service discovery).
- Problem-solving approach grounded in experimentation and data analysis.
- Willingness to expand expertise across the full technical stack.
- Experience building navigation guidance solutions with rigorous quality assurance.
- Comfort working beyond current expertise areas to develop new competencies.

## Technologies
- C++ (primary).
- Rust (acceptable alternative).
- Linux server applications.
- Geospatial data processing.
- Navigation systems.
